## Kubernetes Security Context
Kubernetes Security Context Workbook, organized by namespace and broken into clear, progressive examples with YAML and test commands. We'll cover:
1. Baseline (No Security Context)

2. Pod & Container-Level Security Context

3. Seccomp (RuntimeDefault & Localhost)

4. AppArmor (Ubuntu/Debian only)

5. Capability Hardening

6. Privileged Pods & Restricted Policies

7. Namespace Isolation (optional grouping)

Let’s begin with Part 1–4 below.
